# work projects

A program for generating shifts according to certain rules.

## How to test

Create a toml file and write the following content into it:

test.toml
```toml
[A]
staff = [
    "nameA0",
    "nameA1",
    "nameA2",
    "nameA3",
]

[B]

staff = [
    "nameB0",
    "nameB1",
    "nameB2",
    "nameB3",
    "nameB4",
    "nameB5",
]
```
